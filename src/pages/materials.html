Recursos oficiais

Entenda o Framework Frontend de ponta a ponta

Este hub reúne guias práticos, snippets e checklists para você construir e manter projetos com o Framework Frontend sem depender de ferramentas externas.

1. Primeiros passos

Baixe e abra

Clone o repositório ou baixe o ZIP, abra o index.html em qualquer servidor estático e já navegue.

  • • Sem dependências
  • • Bootstrap, Swiper, Icons já inclusos
  • • Funciona offline
Customize o conteúdo

Abra /cms e edite hero, cards, depoimentos e CTA. Tudo vai direto para o localStorage.

  • • Exportação em JSON
  • • Reset para default
  • • Atualização instantânea
Publique

Envie a pasta para qualquer serviço de hospedagem estática (Vercel, Netlify, S3, FTP tradicional).

  • • Sem CI/CD obrigatório
  • • Cache control em CDN recomendado
  • • Pode rodar dentro de CMSs headless

2. Arquitetura em camadas

Core

Responsável por roteamento, carregamento de páginas, skeleton e helpers globais.

core.jsRouter + estado + fetchAPI
component.jsClasse base para componentes JS
skeleton.jsPlaceholder universal
Conteúdo + Serviços

Tudo que muda com frequência fica desacoplado em src/data e src/services.

  • ContentService (localStorage + eventos)
  • cms-content (seed utilizado no primeiro acesso)
  • blog-posts (dados mockados para demos)

Chamada à API
async function loadReleases() {
  try {
    const response = await core.fetchAPI('releases');
    core.setData('release-list', template(response.data));
  } catch (error) {
    console.error(error);
  }
}
